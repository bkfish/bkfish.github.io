1、1、re.match()函数

函数语法：

函数参数说明：

  • pattern：匹配的正则表达式
  • string：要匹配的字符串
  • flag：标志位，用于控制正则表达式的匹配方式（是否匹配大小写、多行匹配等）
    作用：match()函数只在字符串的开始位置尝试匹配正则表达式，即从位置0开始匹配。如果匹配成功，则返回一个匹配的对象；如果字符串开始不符合正则表达式，则匹配失败，函数返回None。
