切换版本

通过以下命令获取测试环境代码：

将 composer.json 文件的 require 字段设置成如下：

"require": {
    "php": ">=5.6.0",
    "topthink/framework": "5.1.x"
}
```,
然后执行 composer update,获取5.1.x
